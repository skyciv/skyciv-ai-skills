Modeling assumptions
  • Posts are modeled as fixed-base cantilevers into the concrete slab (rigid base) - no footing/anchor/concrete design check is performed, only the AS/NZS 1664 post member check
  • Glass is a non-structural load path only: wind pressure on its tributary area (post spacing x glass height) is transferred to the two adjacent posts as a distributed load - the glass itself is not modeled or checked
  • Wind design pressure = worst of the 4 principal-direction windward wall pressures (AS 1170.2, via the load generator) at the balustrade's elevation, applied both outward and inward on the posts (pressure + suction)
  • AS 1170.1 balustrade line/point loads are indicative defaults, editable per classification - verify against the current edition before final design. Applied as separate (non-concurrent) load cases
  • Post effective-length factor kz = ky = 2.0 (fixed-base/free-top cantilever), unbraced length = post height
  • Only AS/NZS 1170.0 strength load combinations are checked (no serviceability/deflection combos)
  • Number of posts evenly divides the run length so actual spacing never exceeds the target spacing
  • Handrail section and post/handrail alloy-temper are fixed defaults (not user inputs) - shown read-only above
