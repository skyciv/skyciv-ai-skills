Modeling assumptions
  • Panel count auto-chosen for ~4 ft bottom-chord panels (4-10 panels), apex centered at mid-span
  • Chords are continuous through panel points (moment-carrying); webs are pinned (axial only)
  • Every node is out-of-plane braced (purlin/sheathing bracing assumption); left support = pin, right = roller
  • Loads are psf of horizontal plan area, converted to a sloped-member line load via truss spacing x cos(roof slope)
  • 5 ASCE 7-22 §2.3.1 LRFD combinations checked (no floor live load in a roof-only truss): 1.4D+1.4SW; 1.2D+1.2SW+1.6S; 1.2D+1.2SW+1.6S+0.5W; 1.2D+1.2SW+1.0W+0.5S; 0.9D+0.9SW+1.0W (uplift)
  • Every member is checked against the NDS 2018 Wood Member Design calculator using the worst-case (enveloped) force per component across all 5 combinations
  • Checks strength only (bending, shear, tension, compression, combined) - deflection serviceability is not checked, since it needs each member's actual computed deflection under load, not a placeholder
  • "Optimize Section" re-runs the strength check for every section size in the dropdown and recommends the smallest (by area) that passes every member
  • Unbraced length L = member length; effective length for bending Le = 2.06 x L (the calculator's own conservative default)
